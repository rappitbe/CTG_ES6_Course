Destructuring
Destructuring allows you to bind a set of variables to a corresponding set of values anywhere that you can normally bind a value to a single variable.

Destructuring

Thoughts

  • Normally we construct
  • Destructing has always been manual
  • New syntax
  • Pattern Matching

Destructuring

Two Approaches

  • Destructuring Objects
  • Destructuring Arrays

Let's talk about them in that order

Destructuring Objects

Basic Example


  let {city, state, zip} = getAddress();

  log(city);  //"Salt Lake City"
  log(state); //"UT"
  log(zip);   //84115




  function getAddress(){
    return {
      city: "Salt Lake City",
      state: "UT",
      zip: 84115
    };
  }

  function displayPerson(p){
    var name = p.name;
    var age = p.age;
    //do something with name and age to display them
  }

Destructuring Objects

The New Way 


  var person = {name: "Aaron", age: 35};
  displayPerson(person);

  function displayPerson(p){
    let {name, age} = p;
    //do something with name and age to display them
  }


  //No Notes

Destructuring Objects

The New Way - Improved 


  var person = {name: "Aaron", age: 35};
  displayPerson(person);

  function displayPerson({name, age}){
    //do something with name and age to display them
  }


  //No Notes

Destructuring Objects

The New Way - More Improved 


  var person = {name: "Aaron", age: 35};
  displayPerson(person);

  function displayPerson({name = "No Name Provided", age = 0}){
    //do something with name and age to display them
  }


  //No Notes

Destructuring Objects

Nested 

  let person = {
    name: "Aaron",
    age: "35",
    address: {
      city: "Salt Lake City",
      state: "UT",
      zip: 84115
    }
  };

  let {name, age, address: {city, state, zip}} = person;

  //No Note
